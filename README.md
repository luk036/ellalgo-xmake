# 🏉 ellalgo-xmake

> Ellipsoid Method C++ Code

## Features

- At most one square-root per iteration.
- Support Parallel-Cuts.
- Experimental stable version

## Installation

- Require C++20 or above
- The core ellipsoid method depends only on the `xtensor` module.
